D) Sticky Headergo to table of content↑

This template contains 3 header styles:

  1. Normal Header
  2. Sticky Header
  3. Intelligent Sticky Header
It is very easy to change your header styles. You just need to do single change from JS file. You need to edit site.js file placed in "assets/js" directory. You can find the below line written below. You just need to replace normal with fixed or intelligent. It will affect changes globally. 

$('#header').addClass('normal') // normal, fixed or intelligent

G) Change Layoutgo to table of content↑

You can use this template in 2 layouts.

  1. Boxed Layout
  2. Full Width Layout
The version you have is in full width but if you want it in boxed layout then you need to add a class (class="boxed") in wrapper id. It looks like: 

// Boxed Layout
<div id="wrapper" class="boxed">
// Full Width Layout
<div id="wrapper">

H) Change Skin Colorgo to table of content↑

As this template is coded in LESS so it is easy for you to change the glabal colors of skin from single place. You just need to edit the skin.less file placed in "assets/css/" directory. There at the top you can see the code to change skin color. It looks like: 


@skinColor:#50b9ce; //Change here your theme Color

I) Change Fontgo to table of content↑

As this template is coded in LESS so it is easy for you to change the glabal font of skin from single place. You just need to edit the skin.less file placed in "assets/css/" directory. There at the top you can see the code to change font family. It looks like: 


@fontFamily:'Raleway', sans-serif; //Change here your theme Font family

J) Change Location on Mapgo to table of content↑

In the contact section we have added google map. You can change latitude & longitude according to your company address. There are many resources available on internet to get latitude & longitude of any address but for your reference you can use http://www.latlong.net/. Once you have these information then you can edit site.js file & change the existing one to yours. Search for the below code & replace latitude, longitude there. 


google.maps.LatLng(41.236859, -73.667425)

L) Change Favicongo to table of content↑

The easiest way is to show you. Bookmark this page by clicking here. If you reload the page the icon should display in your browsers address field in a similar way to the illustration.

A favicon (short for "favorites icon"), also known as a page icon or an urlicon, is an icon associated with a particular website or webpage. A web designer can create such an icon, and many recent web browsers can then make use of them. Browsers that support favicons may display them in the browser's URL bar, next to the site's name in lists of bookmarks, and next to the page's title in a tabbed document interface.

You can use online websites to generate your favicon. Here is one most popular website for it http://www.favicon-generator.org. You can generate favicon from here & replace favicon.ico on root of your project.
